Path Finder (Custom Dijkstra's)

Instructions:

  • Start Location: Blue box.
  • End Location: Yellow/Green box.
  • Wall: Red box.
  • Click or/and drag cursor to create walls (unpassable objects).
  • Click on a wall to remove it.
  • Click Start button to find path from starting location to the ending location.
  • Click Restart button to remove all walls and paths.
